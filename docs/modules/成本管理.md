# 成本管理

## 概述
记录和管理每日营销投放成本，支持单条录入和批量导入，自动计算单条线索成本。

## 核心文件
- `client/src/components/CostModal.vue` — 成本录入弹窗
- `server/routes/cost.js` — 后端成本 CRUD + 批量导入
- `server/utils/excel.js` — 解析工具

## 数据模型
```
cost_data
  id          INTEGER PK
  cost_date   TEXT    -- 日期
  platform    TEXT    -- 平台（抖音/小红书）
  amount      REAL    -- 总消耗金额
  lead_count  REAL    -- 获得线索数
  unit_cost   REAL    -- 单条成本（自动计算 = amount / lead_count）
  created_at  TEXT
  UNIQUE(cost_date, platform)
```

## 单条录入
- 字段：日期、平台（下拉）、总消耗（元）、获得线索数
- 同一（日期+平台）已存在则覆盖更新
- 后端 `calcUnitCost()` 自动计算并存储 unit_cost

## 批量导入
- 支持 CSV / XLSX 格式
- 智能识别列名：日期、平台、总消耗、获得线索数
- 拖拽上传或点击选择文件
- 同一（日期+平台）已存在则覆盖更新
- 返回统计：新增/更新/跳过条数

## 历史记录
- 列表展示：日期、平台、总消耗、线索数、单条成本
- 支持 inline 编辑（金额和线索数）
- 支持单条删除

## 通用函数
- `calcUnitCost(amount, leadCount)` — 计算单条成本，保留两位小数
- `upsertCostData({cost_date, platform, amount, lead_count, now})` — 插入或更新记录
