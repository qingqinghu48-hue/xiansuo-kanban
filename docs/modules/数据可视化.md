# 数据可视化

## 概述
看板页面的 6 个数据图表，使用 ECharts 渲染。

## 核心文件
- `client/src/components/ChartSection.vue`

## 6 个图表

### 1. 平台来源分布（饼图）
- 数据：按 `平台` 字段分组计数
- 渲染：ECharts 饼图，`labelLine` 外部标签，自动计算百分比

### 2. 有效性分布（饼图）
- 数据：按 `线索有效性` / `有效性` 字段分组计数
- 渲染：同上

### 3. 抖音每日总消耗（柱状图）
- 数据：`costData` 中平台 = "抖音" 的 `amount`，按日期聚合
- 渲染：渐变色填充 + 圆角顶部 + 顶部数值标签

### 4. 小红书每日总消耗（柱状图）
- 数据：`costData` 中平台 = "小红书" 的 `amount`，按日期聚合
- 渲染：同上

### 5. 抖音单条线索成本（折线图）
- 数据：`costData` 中平台 = "抖音" 的 `unit_cost`，按日期聚合
- 渲染：平滑曲线 + 面积渐变填充 + 数据点标签

### 6. 小红书单条线索成本（折线图）
- 数据：`costData` 中平台 = "小红书" 的 `unit_cost`，按日期聚合
- 渲染：同上

## 横向滚动
柱状图和折线图在数据点较多时自动横向滚动：
- 柱状图：每数据点 40px 宽度
- 折线图：每数据点 50px 间距
- 容器 `overflow-x: auto`
