# 线索管理

## 概述
管理员对招商线索的增删改查操作，入口在管理后台（`/admin`）。

## 核心文件
- `client/src/views/AdminView.vue` — 管理后台主页面
- `client/src/components/EditModal.vue` — 编辑浮窗
- `client/src/components/DetailModal.vue` — 详情浮窗
- `server/routes/leads.js` — 后端线索 CRUD

## 新增线索
- 字段：姓名、手机号、平台（下拉）、所属大区（多选）、线索有效性、线索类型、所属招商、备注、入库日期
- 手机号必填，后端查重（`UNIQUE(phone)`）

## 编辑线索
- 平台改为单选下拉（选项来自 platforms 表）
- 所属大区改为多选下拉，保存时自动用 "、" 拼接
- 小红书平台才显示"用户小红书 ID"字段
- admin 可修改"所属招商"（选项来自启用的 agent 列表）
- 修改手机号时检查是否与其他记录冲突

## 删除线索
- 单条删除：按手机号删除
- 批量删除：勾选多行后批量删除
- **权限限制**：仅 admin 可操作，招商员无删除权限
