# 线索看板

## 概述
看板首页，展示全部招商线索的筛选、统计、图表和列表。

## 核心文件
- `client/src/views/DashboardView.vue` — 页面主容器，串联所有子组件
- `client/src/components/FilterBar.vue` — 筛选栏
- `client/src/components/KpiCards.vue` — 6 个 KPI 统计卡片
- `client/src/components/DataTable.vue` — 线索列表（排序、分页、操作）
- `client/src/components/ChartSection.vue` — 6 个数据图表

## 数据流
1. `DashboardView.onMounted()` 拉取全部线索 `allData` 和成本数据 `costData`
2. `FilterBar` 发出筛选条件 `{ds, de, fp, fv, flt, fr, fs, fk}`
3. `DashboardView.onFilter()` 同步筛选线索数据 → `filtered`，成本数据 → `filteredCostData`
4. `KpiCards` / `ChartSection` / `DataTable` 分别消费 `filtered` 和 `costData`

## 筛选逻辑
| 条件 | 字段 | 说明 |
|------|------|------|
| 日期范围 | `入库日期` | 起止日期闭区间 |
| 平台 | `平台` | 精确匹配 |
| 有效性 | `线索有效性` / `有效性` | 兼容新旧字段名 |
| 线索类型 | `线索类型` | 广告线索 / 自然流线索 |
| 大区 | `所属大区` | 支持多选，按 "、" 拆分后匹配 |
| 招商员 | `所属招商` | 精确匹配 |
| 关键词 | `姓名`+`手机号`+`所属大区`+`所属招商` | 模糊搜索 |

## KPI 统计（6 个板块）
1. **总线索数** — `filtered.length`
2. **抖音线索数** — 平台 = "抖音" 的数量 + 占比
3. **小红书线索数** — 平台 = "小红书" 的数量 + 占比
4. **其他平台线索数** — 排除抖音/小红书后的数量和 + 平台名称列表
5. **累计营销费用消耗** — `filteredCostData` 中 `amount` 总和（admin 可见）
6. **单条线索平均成本** — `amount` 总和 ÷ `lead_count` 总和（admin 可见）

## 列表操作
- 前端分页（每页 20 条）
- 点击行查看详情（`DetailModal`）
- admin 可编辑（`EditModal`）、删除、批量删除
